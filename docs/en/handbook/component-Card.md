## Key Attribute for component

| Attribute Name | Attribute Description                                    | Additional Info        |
| :------------- | :------------------------------------------------------- | :--------------------- |
| Source Key     | Component ID for the data source                         | N/A for this component |
| Title          | Title for the divider                                    |                        |
| Padding        | Set the padding width for the parameters within the card |                        |
| Show Header    | Config if need to show the header (title) or not         |                        |
| Bordered       | Set if need to display the border for the card           |                        |
| Shadow         | Set the behavior for shadow effect of the card           |                        |
