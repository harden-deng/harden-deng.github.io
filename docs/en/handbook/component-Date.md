## Key Attribute for component

| Attribute Name | Attribute Description                                                                                                                                                                                                                                                                       | Additional Info                                           |
| :------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :-------------------------------------------------------- |
| Source Key     | Component ID for the data source                                                                                                                                                                                                                                                            | Config when the input value is comes from other component |
| Display Type   | Control the display type for the date selection,`<br>`the component will act differently based on the display type selected                                                                                                                                                               |                                                           |
| Help Text      | Help text for the content                                                                                                                                                                                                                                                                   |                                                           |
| Value Type     | Choose the limitation type for the date selection range,`<br>`when it is “Absolute Value”, you can select specific date for the Min/Max value,`<br>` when it is “Relative Value”, you need to set relative value for the Min/Max, `<br>`the date range will be dynamic calculated |                                                           |
| Max Value      | Set the max value for the date range,`<br>`leave it blank if you don’t need to control the range                                                                                                                                                                                         |                                                           |
| Min Value      | Control the offset width for the current column (whole grid is 24 spans)                                                                                                                                                                                                                    |                                                           |
| Format         | Control display format for the date selected, default is YYYY-MM-DD                                                                                                                                                                                                                         |                                                           |
| Default Value  | Default value for the component when page is loaded                                                                                                                                                                                                                                         |                                                        |
