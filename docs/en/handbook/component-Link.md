## Key Attribute for component

| Attribute Name  | Attribute Description                                                                                                     | Additional Info                                           |
| :-------------- | :------------------------------------------------------------------------------------------------------------------------ | :-------------------------------------------------------- |
| Source Key      | Component ID for the data source                                                                                          | Config when the input value is comes from other component |
| Destination URL | Set the address for the destination URL                                                                                   |                                                           |
| Query String    | Set the query string which need to add to the URL, multiple and dynamic value for the query string parameter is supported |                                                           |
| Open Mode       | New Page: jump to the link in a new page mode<br/>Current Page: jump to the link in current page |                                                           |
