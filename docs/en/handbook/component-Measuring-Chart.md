## Key Attribute for component

| Attribute Name       | Attribute Description                                                                                                                                      | Additional Info        |
| :------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------- |
| Source Key           | Component ID for the data source                                                                                                                           | N/A for this component |
| Block Select Options | Set the data selection range for the block, can support static & dynamic data source, if no option is configured, the block would be display as a text box |                        |
| Measure Points List  | Set the data selection range for the Measure Points, can support static & dynamic data source                                                              |                        |
| Single Tolerance     | Title for the item row                                                                                                                                     |                        |
| Toolbar Visible      | Control if need to display the toolbar                                                                                                                     |                        |
| Block Editable       | Control if control the block to be only from dropdown list                                                                                                 |                        |
| Block Size           | Default measurement point size number for each block item                                                                                                  |                        |
| Row Size             | Default row number for the component when the page is loaded                                                                                               |                        |
