## Key Attribute for component

| Attribute Name                    | Attribute Description                                                                 | Additional Info        |
| :-------------------------------- | :------------------------------------------------------------------------------------ | :--------------------- |
| Source Key                        | Component ID for the data source                                                      | N/A for this component |
| Max length                        | Maximum length for the input content, leave it blank if you don't want any limitation |                        |
| Default Value                     | Default value for the text box when page is loaded                                    |                        |
| Autosize                          | Control if the text area box will be auto-adjusted based on the content input         |                        |
| Rows                              | Control the display row size of the text area box                                     |                        |
