## Key Attribute for component

| Attribute Name         | Attribute Description                                            | Additional Info                                             |
| :--------------------- | :--------------------------------------------------------------- | :---------------------------------------------------------- |
| Source Key             | Component ID for the data source                                 | N/A for this component                                      |
| Comment Select Options | Set the data source for the comment drop down list               | Data source configuration can refer to “select” component |
| Comment Select Visible | Set whether show the comment drop down under each photo          |                                                             |
| Multiple               | Set whether to enable upload photo in batch                      |                                                             |
| Toolbar Visible        | Control if need to display the toolbar for the flex table or not |                                                             |
| Rows                   | Set default row number when the flex table is added              |                                                             |
| Columns                | Set default column number when the flex table is added           |                                                             |
