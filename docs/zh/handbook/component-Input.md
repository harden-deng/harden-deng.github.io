## Key Attribute for component

| Attribute Name                    | Attribute Description                                                           | Additional Info                                           |
| :-------------------------------- | :------------------------------------------------------------------------------ | :-------------------------------------------------------- |
| Source Key                        | Component ID for the data source                                                | Config when the input value is comes from other component |
| Max length                        | Maximum length for the input content                                            |                                                           |
| Default Value                     | Default value for the text box when page is loaded                              |                                                           |
| Show clear button                 | Control whether to show the clear button for the text box                       |                                                           |
| Show password                     | Control whether to display the text box input area as password                  |                                                           |
