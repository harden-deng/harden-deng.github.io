# Report Template Designer

## Desinger Introduction

The designer is divided into the following areas:

![img](images/Designer_fullPic.png)

### (1) Object Switcher 

Click to see the form template field structure tree.

### (2) Terminal Adapter

Click to switch to different canvas while design the report template.

### (3) Components 

Different types of components which can be added/drag to the canvas to design the report.

### (4) From Area

Web canvas, main area for designers to design the report.

### (5) Attrubites

Can switch to the attribute configuration for Component Level and whole Form Level.

### (6) Field Properties Panel

Attribute configuration for the selected component, which can be used to set the component display format, behavior and data source.

### (7) Operating Panel

* **Publish:** click to publish the current template to template library.
* **Save:** click to save the current form template as draft.
* **Preview:** click to preview the current design of the report.
