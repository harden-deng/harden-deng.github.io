## Key Attribute for component

| Attribute Name | Attribute Description                                        | Additional Info        |
| :------------- | :----------------------------------------------------------- | :--------------------- |
| Source Key     | Component ID for the data source                             | N/A for this component |
| Size (MB)      | Control the maximum size control for the video uploaded      |                        |
| Width (px)     | Set the width for the thumbnail after the video is uploaded  |                        |
| Height (px)    | Set the height for the thumbnail after the video is uploaded |                        |
