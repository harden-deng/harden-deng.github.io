## Key Attribute for component

| Attribute Name | Attribute Description                                                             | Additional Info                                           |

| Minimum        | Set the min value for the slider bar range                                        |                                                           |
| Maximum        | Set the max value for the slider bar range                                        |                                                           |
| Step           | Set the number duration of the interval between each drag point of the slider bar |                                                           |
